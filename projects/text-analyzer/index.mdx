## Your first AI project

Build a text analysis tool that uses AI techniques - without needing any API keys! This project brings together everything you've learned.

## What you'll build

A command-line tool that can:
- Analyze sentiment (positive/negative)
- Extract key phrases
- Summarize text
- Count words and readability

All using free, local Python libraries!
