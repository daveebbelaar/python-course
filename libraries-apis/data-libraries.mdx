## Python for data

Python excels at data processing thanks to powerful libraries. These tools help you work with spreadsheets, databases, and large datasets efficiently.

The main libraries:
- **pandas**: Work with tables (like Excel)
- **NumPy**: Fast numerical operations
- **matplotlib**: Create charts and graphs

## Pandas basics

Pandas makes data manipulation easy:

```python
import pandas as pd

# Create a DataFrame (like a spreadsheet)
data = {
    "name": ["Alice", "Bob", "Charlie"],
    "age": [25, 30, 35],
    "city": ["NYC", "LA", "Chicago"]
}

df = pd.DataFrame(data)
print(df)

# Output:
#       name  age      city
# 0    Alice   25       NYC
# 1      Bob   30        LA
# 2  Charlie   35   Chicago
```

### Common operations

```python
# Read CSV file
df = pd.read_csv("data.csv")

# Basic info
print(df.head())      # First 5 rows
print(df.info())      # Column types
print(df.describe())  # Statistics

# Filter data
adults = df[df["age"] >= 18]
nyc_residents = df[df["city"] == "NYC"]

# Add column
df["birth_year"] = 2024 - df["age"]

# Group and aggregate
avg_age_by_city = df.groupby("city")["age"].mean()
```

<Tip>
  Pandas can read many formats: CSV, Excel, JSON, SQL databases, and more!
</Tip>

## Working with CSV files

```python
import pandas as pd

# Read CSV
df = pd.read_csv("sales.csv")

# Quick analysis
total_sales = df["amount"].sum()
avg_sale = df["amount"].mean()
top_product = df.groupby("product")["amount"].sum().max()

# Save results
results = pd.DataFrame({
    "metric": ["total", "average", "top_product"],
    "value": [total_sales, avg_sale, top_product]
})

results.to_csv("analysis_results.csv", index=False)
```

## NumPy arrays

NumPy provides fast numerical operations:

```python
import numpy as np

# Create arrays
numbers = np.array([1, 2, 3, 4, 5])
matrix = np.array([[1, 2], [3, 4]])

# Math operations (very fast!)
squared = numbers ** 2
mean = np.mean(numbers)
std = np.std(numbers)

# Array operations
reshaped = numbers.reshape(5, 1)
transposed = matrix.T
```

## Basic visualization

Create charts with matplotlib:

```python
import matplotlib.pyplot as plt
import pandas as pd

# Sample data
df = pd.DataFrame({
    "month": ["Jan", "Feb", "Mar", "Apr"],
    "sales": [100, 120, 140, 110]
})

# Bar chart
plt.bar(df["month"], df["sales"])
plt.title("Monthly Sales")
plt.xlabel("Month")
plt.ylabel("Sales ($)")
plt.show()

# Line chart
plt.plot(df["month"], df["sales"], marker="o")
plt.title("Sales Trend")
plt.show()
```

## Common patterns

### Data cleaning
```python
# Remove missing values
df_clean = df.dropna()

# Fill missing values
df["age"].fillna(df["age"].mean(), inplace=True)

# Remove duplicates
df_unique = df.drop_duplicates()
```

### Data analysis
```python
# Quick statistics
print(df["price"].describe())

# Correlations
correlation = df.corr()

# Value counts
print(df["category"].value_counts())
```

## Common mistakes

<AccordionGroup>
  <Accordion title="Not installing libraries">
    ```python
    # Wrong - forgot to install
    import pandas  # ModuleNotFoundError
    
    # Right - install first
    # Run: pip install pandas numpy matplotlib
    import pandas as pd
    ```
  </Accordion>

  <Accordion title="Modifying without copy">
    ```python
    # Wrong - modifies original
    filtered = df[df["age"] > 30]
    filtered["age"] = 0  # Warning!
    
    # Right - use .copy()
    filtered = df[df["age"] > 30].copy()
    filtered["age"] = 0
    ```
  </Accordion>

  <Accordion title="Large file memory issues">
    ```python
    # Wrong - loads entire file
    df = pd.read_csv("huge_file.csv")
    
    # Right - read in chunks
    for chunk in pd.read_csv("huge_file.csv", chunksize=1000):
        process(chunk)
    ```
  </Accordion>
</AccordionGroup>
