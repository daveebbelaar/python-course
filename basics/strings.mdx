## What are strings?

Strings are sequences of characters - basically text. Whenever you work with names, messages, URLs, or any text data, you're using strings.

In Python, strings are incredibly powerful. You can slice them, combine them, search through them, and transform them in countless ways.

## Creating strings

Three ways to create strings:

```python
# Single quotes
name = 'Alice'

# Double quotes
message = "Hello, World!"

# Triple quotes for multiple lines
poem = """Roses are red,
Violets are blue,
Python is awesome,
And so are you!"""
```

<Tip>
  Use single or double quotes consistently. I prefer double quotes because they let you include apostrophes: `"It's Python!"`
</Tip>

## String operations

### Concatenation (joining)

```python
first_name = "Jane"
last_name = "Doe"

# Using +
full_name = first_name + " " + last_name  # "Jane Doe"

# Using f-strings (modern Python way!)
greeting = f"Hello, {first_name}!"  # "Hello, Jane!"

# Multiple variables
age = 25
intro = f"I'm {first_name} and I'm {age} years old"
```

### Repetition

```python
# Repeat strings
star = "*"
stars = star * 10  # "**********"

# Create separators
separator = "-" * 50
print(separator)
print("TITLE")
print(separator)
```

## String methods

Strings come with powerful built-in methods:

### Changing case

```python
text = "Python Programming"

print(text.lower())      # "python programming"
print(text.upper())      # "PYTHON PROGRAMMING"
print(text.title())      # "Python Programming"
print(text.capitalize()) # "Python programming"
print(text.swapcase())   # "pYTHON pROGRAMMING"
```

### Cleaning strings

```python
messy = "  hello world  \n"

print(messy.strip())     # "hello world" (removes whitespace)
print(messy.lstrip())    # "hello world  \n" (left only)
print(messy.rstrip())    # "  hello world" (right only)

# Remove specific characters
price = "$19.99"
print(price.strip("$"))  # "19.99"
```

### Finding and replacing

```python
message = "I love Python programming with Python"

# Check if something exists
print("Python" in message)        # True
print(message.startswith("I"))   # True
print(message.endswith("Python")) # True

# Find position
print(message.find("Python"))     # 7 (first occurrence)
print(message.count("Python"))    # 2 (number of times)

# Replace
new_message = message.replace("Python", "JavaScript")
print(new_message)  # "I love JavaScript programming with JavaScript"
```

## String indexing and slicing

Strings are sequences, so you can access individual characters:

```python
text = "Python"

# Indexing (starts at 0)
print(text[0])    # "P"
print(text[1])    # "y"
print(text[-1])   # "n" (last character)
print(text[-2])   # "o" (second to last)

# Slicing [start:end:step]
print(text[0:3])  # "Pyt" (characters 0,1,2)
print(text[2:5])  # "tho"
print(text[:3])   # "Pyt" (start to 3)
print(text[3:])   # "hon" (3 to end)
print(text[::2])  # "Pto" (every 2nd character)
print(text[::-1]) # "nohtyP" (reversed!)
```

## String formatting

### f-strings (recommended)

```python
name = "Alice"
age = 30
height = 5.6

# Basic formatting
print(f"Name: {name}, Age: {age}")

# Expressions in f-strings
print(f"In 10 years, {name} will be {age + 10}")

# Formatting numbers
price = 19.99
print(f"Price: ${price:.2f}")  # "Price: $19.99"

# Padding and alignment
print(f"{name:>10}")  # "     Alice" (right align)
print(f"{name:<10}")  # "Alice     " (left align)
print(f"{name:^10}")  # "  Alice   " (center)
```

## Escape characters

Special characters in strings:

```python
# Common escape characters
print("Hello\nWorld")     # New line
print("Hello\tWorld")     # Tab
print("She said \"Hi\"")  # Quote inside quotes
print("C:\\Users\\Name")  # Backslash
print("Line 1\nLine 2\nLine 3")

# Raw strings (no escaping)
path = r"C:\Users\Name\Documents"
print(path)
```

## Common mistakes

### Modifying strings
```python
# Strings are immutable (can't be changed)
name = "Alice"
# name[0] = "B"  # Error!

# Create a new string instead
name = "B" + name[1:]  # "Blice"
```

### Forgetting f in f-strings
```python
# Wrong
name = "Alice"
message = "Hello {name}"  # Prints: "Hello {name}"

# Right
message = f"Hello {name}"  # Prints: "Hello Alice"
```

## String methods reference

```python
text = "hello world"

# Checking
text.isalpha()    # False (has space)
text.isdigit()    # False
text.isspace()    # False
text.islower()    # True
text.isupper()    # False

# Splitting and joining
words = text.split()           # ["hello", "world"]
rejoined = " ".join(words)     # "hello world"

# More methods
text.center(20, "*")           # "****hello world*****"
text.ljust(20, "-")           # "hello world---------"
text.partition(" ")           # ("hello", " ", "world")
```
